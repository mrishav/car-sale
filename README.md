# McLaren 570S For Sale Website

**Description**  
Website designed to help sell a McLaren 570S.  
Feel free to use this template if you’re looking to sell your own car.

## Website Features
- Modern glass-morphism design
- Full-screen hero section
- Interactive photo gallery
- Vehicle specifications and features
- Contact section with direct email and phone links

## Tech Stack
- React 19
- Vite 7
- TailwindCSS v4
- Lucide React

## Development
```bash
npm install
npm run dev
npm run build
